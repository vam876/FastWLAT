## 升级说明

### 从旧版本升级

1. 备份数据
2. 下载新版本
3. 安装新版本
4. 导入备份数据

### 数据库更新

GeoIP 数据库建议每季度更新一次：
1. 从 MaxMind 下载最新数据库
2. 替换 `data/` 目录中的文件
3. 重启应用
